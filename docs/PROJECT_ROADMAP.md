# Project Philosophy

Throughout this project:

* Business understanding comes before technical implementation.
* Every technical task must support a business objective.
* Documentation is maintained alongside development.
* Git commits are made regularly.
* Code quality, readability, and maintainability are priorities.
* The goal is to think and work like a professional analytics consultant rather than simply completing technical exercises.
