# E-Commerce Business Intelligence & Profitability Analysis

## Overview

This project simulates a real-world analytics consulting engagement for **Olist**, a Brazilian e-commerce marketplace.

The objective is to identify the key business factors affecting profitability, customer satisfaction, seller performance, and operational efficiency using SQL, PostgreSQL, Python, and Power BI.

---

## Project Objectives

* Understand the business problem
* Analyze customer behavior
* Evaluate product performance
* Measure seller performance
* Investigate delivery efficiency
* Build executive dashboards
* Deliver data-driven business recommendations
